# Indexing Components

Komponenty pre správu a optimalizáciu indexácie stránok v Google Search Console.

## Súbory:
- `IndexingFixCenter.tsx` - Hlavný nástroj na opravu indexácie s konkrétnymi krokmi
- `IndexingAccelerator.tsx` - Existujúci komponent pre urýchlenie indexácie

## Použitie:
```tsx
import IndexingFixCenter from './indexing/IndexingFixCenter';

<IndexingFixCenter />
```
